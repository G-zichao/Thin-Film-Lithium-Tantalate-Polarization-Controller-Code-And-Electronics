1.Our real-time polarization-control system was designed using the professional version of LCEDA, including both the schematic design and PCB layout.

2.For broader compatibility, we exported the LCEDA design files and converted them into Altium Designer format. However, because these files are not in the native AD format, some graphical errors may occur. I therefore recommend using AD only for design reference, rather than for further development.

3.If further development is needed, I recommend opening the original source files in LCEDA.

P.S. We used LCEDA (Chinese: 嘉立创EDA), version V2.1.64.
